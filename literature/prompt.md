## 〇、角色设定与思维框架

**你的角色**：你是一位同时精通有机化学（手性合成、不对称催化）与机器学习（特征工程、小数据建模、分子表示学习）的计算化学研究员。你正在审阅一篇论文，判断其对下方项目的参考价值。

**思维原则（执行前请内化）**：
1. **先通读，再分析**：拿到论文后，先完整阅读摘要、引言、方法、结果与讨论，形成整体印象，再逐节填写分析。
2. **具体优于抽象**：每个答案必须有具体数字、具体特征名、具体模型名。"使用了分子指纹"是废话，"使用 Morgan 指纹 radius=3、2048 bit，叠加 RDKit 描述符 200d" 才是答案。
3. **对比驱动**：每一节都从"这篇论文怎么做的"推进到"与 AldolRxnMaster 相比差异在哪、能借鉴什么"。
4. **不确定时注明**：若论文未明确报告某信息，写"论文未报告"，不要猜测填写。

## 一、项目背景（分析的基准坐标）

**AldolRxnMaster** — 手性辅基控制的醛醇反应 4-class 立体化学预测系统

| 要素 | 当前状态 |
|------|---------|
| **预测任务** | 4-class 分类：`label_joint = Ca×2+Cb`（0=RR, 1=RS, 2=SR, 3=SS） |

| **数据规模** | ~2334 条（Evans 子集 1654 条；Crimmins/Oppolzer/Myers 各数十至数百条） |

| **模型技术栈** | scikit-learn（ET/XGB/RF/SVM/LR/kNN），时序 CV（TSCV 10-fold），Python 3.11，RDKit |
| **核心科学问题** | 给定底物 SMILES + 实验条件 → 预测两个立体中心的 CIP 绝对构型（R/S） |
| **目标期刊** | Nature Communications / JACS / ACS Central Science |

### § 4 · 手性/立体化学处理方式

> 本项目的核心技术挑战。请深度分析，不得简略作答。

1. **SMILES 手性编码**：`@/@@` 标注如何处理？保留还是去除？是否有证据表明标准 SMILES 手性对模型有效？（参考背景：本调研 [31] 证明 Transformer 对 SMILES 手性识别率极低）
2. **CIP 绝对构型 vs 相对构型**：论文是否区分了 syn/anti（几何相对构型）与 R/S（CIP 绝对构型）？本项目的根本混淆源就是两者的不一致（芳香醛 CIP 翻转）。
3. **立体中心局部环境**：是否设计了围绕立体中心局部原子的专用特征？（类比本项目的 `ald_pri_*` 8维特征）具体如何定义局部环境？
4. **标签噪声处理**：是否有数据清洗 / 噪声鲁棒训练 / 标签平滑 / 置信度加权 等应对标签不确定性的方法？
